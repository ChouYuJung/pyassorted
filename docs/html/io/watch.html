watch¶

pyassorted.io.watch.watch(filepath: ~typing.Union[~pathlib.Path, str], period: float = 0.1, raise_timeout_errors: ~typing.Tuple = (<class 'TimeoutError'>,)) → Generator[Path, None, None]¶

Watch a file for changes.

Parameters¶

filepathPathText

Path to file to watch.

periodfloat, optional

period to check for changes, by default 0.1

Yields¶

PathText

Path to file that has changed.

Examples¶

>>> from pyassorted.io import watch
>>> for filepath in watch('file.txt'):
...     print(filepath)

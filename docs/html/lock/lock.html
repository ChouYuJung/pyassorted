lock.lock¶

FileLock¶

pyassorted.lock.filelock.FileLock(file_name: str | None = None, timeout: float = 10, delay: float = 0.05, lock_expire: int = 60)¶

Soft File lock. The lock uses a file to store the lock status.

Examples¶

>>> from concurrent.futures import ThreadPoolExecutor
>>> from pyassorted.lock import FileLock
>>>
>>> number = 0
>>> workers = 40
>>> tasks = 100
>>> lock = FileLock()
>>>
>>> def add_one():
...     global number
...     with lock:
...         number += 1
>>>
>>> with ThreadPoolExecutor(max_workers=workers) as executor:
...     futures = [executor.submit(add_one) for _ in range(tasks)]
...     for future in futures:
>>>         future.result()
>>>
>>> assert number == tasks

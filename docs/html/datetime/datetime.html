Timer¶

pyassorted.datetime.Timer()¶

The Timer class is a context manager that measures elapsed time.

Examples¶

>>> import time
>>> from pyassorted.datetime import Timer
>>>
>>> timer = Timer()
>>> timer.click()
>>> time.sleep(1)
>>> timer.click()
>>> print(round(timer.read()))  # 1
>>>
>>> with timer:
...     time.sleep(1)
>>> print(round(timer.read()))  # 1
